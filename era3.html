Era 3 Architectural Review

Smart Brain, Hard-Wired Limbs.

1. Forced Schemas

The LLM doesn't "discover." Every professional tool is hard-stuffed into the prompt. No dynamic handshake.

2. The Reroute Prison

The AI defines the chain, but the developer must manually code the rerouting bridge for every single tool in that chain.

3. CONTRACT FRAGILITY

If one API field changes, the bridge crashes. The system lacks a protocol; it only has a rigid contract.
