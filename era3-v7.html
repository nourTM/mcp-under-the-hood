Step 1

Function Calling Is Different

Same tool idea. Different model-specific wrappers.

OpenAI style
{
  "tool_call": {
    "name": "detect",
    "arguments": {
      "asset_url": "..."
    }
  }
}
Anthropic style
{
  "type": "tool_use",
  "name": "detect",
  "input": {
    "asset_url": "..."
  }
}
Google style
{
  "functionCall": {
    "name": "detect",
    "args": {
      "asset_url": "..."
    }
  }
}

One tool concept. Different calling conventions.
